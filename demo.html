form-tools.js

Clone pattern plugin

Example: Simple Clone

Structure:

			
<form cp-init="userList">
	<span cp-add>Clone</span>
	<div cp-viewportport>
		<div cp-template>
			<div cp-inputs>
	  			<label cp-counter>1</label>
	    			<input type="text" name="name[]">
	    			<input type="email" name="email[]">
	    			<span cp-rm></span>
			</div>
		</div>
	</div>
</form>

Clone

Example: Clone With Options

Options:

  • minNode - Min. positions = 1; minNodeMsg - Personal reject message;
  • maxNode - Max. positions = 6; maxNodeMsg - Personal reject message;
  • massAdd - Adition via button hold of 5 positions;
  • startNode - Start build with 3 positions;

Structure:

			
<form cp-init="addressList" cp-options="{
	minNode:1, minNodeMsg:'This form must contain at least one address.',
	maxNode:6, maxNodeMsg:'You can add only 6 adresses.', massAdd:5, startNode:3
}">
	<span cp-add>Clone</span>
	<div cp-viewport>
		<div cp-template>
			<div cp-inputs>
	  			<label cp-counter>1</label>
	    			<input type="text" name="country[]">
	    			<input type="text" name="city[]">
	    			<input type="text" name="street[]">
	    			<span cp-rm></span>
			</div>
		</div>
	</div>
</form>

Result:

Clone

Example: Clone With Sepecified Data

Structure:

			
<form cp-init="orders" cp-options="{massAdd:5}">
	<span cp-add>Clone</span>
	<div cp-viewport>
		<div cp-template>
			<div cp-inputs>
	  			<label cp-counter>1</label>
	    			<input type="text" name="name[]">
	    			<input type="text" name="cost[]">
	    			<input type="datetime-local" name="date[]">
	    			<span cp-rm></span>
			</div>
		</div>
	</div>
</form>

Putting data (json):

			
<script type="text/javascript">
	formTools.pushData('orders',[
		['blackberry', '15$', '2017-06-19T09:30'],
		['', '42$', '2017-06-07T11:22'],
		['strawberry', '21$', '2017-05-22T00:15'],
		['blueberry', '18$', '2017-02-01T18:35'],
		['cranberries', '14$', '2016-11-07T21:04']
	]);
</script>

Result:

Clone

If you need to render an add button or viewport from the initialization area, you must set an identifier equal to the value of the cp-init tag.

			
<span cp-add="imInitId">Clone</span>
	...
<form cp-init="imInitId">
	...
</form>
	...
<div cp-viewport="imInitId">
	...
</div>

Mark input tag via set-nulls attribute when you need adding '.00' after your numbers.

			
<input type="number" set-nulls/>
